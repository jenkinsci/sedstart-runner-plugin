# sedstart-runner

## Introduction

This plugin allows you to run tests created on platform app.sedstart.com

## Getting started

1. Generate API Key for your project in sedstart platform under API Keys Section
2. Store this API Key as System Level Environment variable with any name say SEDSTART_API_KEY
3. pass this name of environment variable to one of the inputs
4. For rest of the input parameters you can copy from copy icon from sedstart Run Button
